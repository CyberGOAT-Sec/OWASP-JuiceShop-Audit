# 🛡️ Аудит безопасности OWASP Juice Shop

Проект по поиску и эксплуатации уязвимостей в рамках формирования портфолио специалиста по ИБ (**CyberGOAT-Sec**).

## 🎯 Цель исследования
Проведение аудита безопасности веб-приложения Juice Shop, выявление критических уязвимостей и демонстрация методов их эксплуатации.

## 🛠️ Инструментарий
* **ОС:** Kali Linux
* **Браузер:** Firefox (Инструменты разработчика)
* **Разведка:** Dirsearch (fuzzing директорий)
