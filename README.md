# 🛡️ Аудит безопасности OWASP Juice Shop

Проект для моего профессионального портфолио (CyberGOAT).

## 🎯 Цель
Поиск и эксплуатация веб-уязвимостей в тренировочной среде Juice Shop для отработки навыков пентеста.

## 🛠️ Инструменты
* **Kali Linux** (в Docker-контейнере)
* **Firefox Dev Tools**
* **Инструменты разведки (Dirsearch)**

## 📁 Наденные уязвимости

### 1. SQL-инъекция (Обход авторизации)
* **Суть:** Недостаточная фильтрация данных в форме логина.
* **Результат:** Удалось зайти под аккаунтом администратора без пароля.
![Вход под админом](screenshots/01_sql_injection_admin.jpg)

### 2. Межсайтовый скриптинг (Reflected XSS)
* **Суть:** Внедрение скрипта через строку поиска.
* **Результат:** Вывод окна alert с куки-файлами пользователя.
![XSS кража куки](screenshots/02_xss_cookie_theft.jpg)

### 3. Утечка конфиденциальных данных (FTP)
* **Суть:** Открытый доступ к директории с бэкапами.
* **Результат:** Получен доступ к папке `/ftp` и скачан файл с купонами.
![Утечка через FTP](screenshots/03_ftp_data_leak.jpg)
